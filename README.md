# **Init the tools -> AI4Industry.**

## Intro.
Ce dépôt forme (un peu) sur les outils de base pour faire de l'IA.

## Requis.
`Docker` est nécessaire pour démarer le tuto.

## C'est parti !!!
Pour démarer rien de plus simple.

- Sur **Windows** :
```bash
python main.py
```

- Sur **Linux** et donc **MacOS** :
```bash
python3 main.py
```
